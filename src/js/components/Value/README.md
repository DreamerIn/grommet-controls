## Value
A single value visualization with icon and label.

## Usage

```javascript
$ npm install grommet-controls

    import { Value } from 'grommet-controls';

    <Value value={...} label={...} />
```

## Properties

**label**

Value label.

```
string
```

**color**

The value color.

```
string
```

**value**

The value.

```
string
element
number
```

**weight**

Font weight. Defaults to `bold`.

```
normal
bold
number
```

**gap**

Spacing between the value and the label. Defaults to `small`.

```
xsmall
small
medium
large
xlarge
```

**size**

Value element size. Defaults to `large`.

```
xsmall
small
medium
large
xlarge
xxlarge
```
