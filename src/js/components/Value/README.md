## undefined
A single value visualization with a label.

## Usage

```javascript
$ npm install grommet-controls

    import { Value } from 'grommet-controls';

    <Value value={...} label={...} />
```

## Properties

**label**

Value label.

```
node
```

**color**

The value color.

```
string
```

**value**

The value.

```
node
```

**weight**

Font weight.

```
normal
bold
number
```

**gap**

Spacing between the value and the label.

```
xsmall
small
medium
large
xlarge
```

**size**

Value element size.

```
xsmall
small
medium
large
xlarge
xxlarge
```
