## undefined
A page header component.
      

## Usage

```javascript

      $ npm install grommet-controls
 
      import { Header } from 'grommet-controls';

      <Header>
        ...
      </Header>  
    
```

## Properties

**position**

Position property for the header element Defaults to `relative`.

```
fixed
absolute
sticky
static
relative
```

**zIndex**

z-index for the header. Defaults to `10`.

```
number
```
